# 02 프로그래밍 핵심 개념 in JavaScript   

## 01. 숫자형

숫자형 자료형은 사칙연산 외에도 다양한 연산 가능

나머지   
```JavaScript
console.log(7 % 3);
```
```
1
```


거듭제곱   
```JavaScript
console.log(2 ** 3);
```
```
8
```

## 03. 문자열 기본
문자열 표현하려면 따옴표로 감싸줘야 함.   

## 04. 문자열 활용
문장 안에 큰따옴표(")나 작은따옴표(') 들어가는 경우   
1. "I'm Ironman."
2. "He said \"I'm Ironman.\""
3. (백틱)He said I'm Ironman.(백틱)   
의 방법으로 입력해서 쓸 수 있음.   
(백틱은 숫자 1 왼쪽의 키)
